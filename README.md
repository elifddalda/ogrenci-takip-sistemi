# 🎓 Öğrenci Takip Sistemi

C# Windows Forms ve SQL Server kullanılarak geliştirilmiş bir öğrenci takip uygulamasıdır.

## 🚀 Özellikler
- Öğrenci ekleme
- Öğrenci listeleme
- Öğrenci silme
- Öğrenci güncelleme

## 🛠 Teknolojiler
- C#
- Windows Forms
- SQL Server

## 💡 Amaç
Temel CRUD işlemlerini öğrenmek amacıyla geliştirilmiştir.
